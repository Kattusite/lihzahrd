lihzahrd.npcs¶

class lihzahrd.npcs.NPC(type_: lihzahrd.npcs.npctype.EntityType, name: str, position: lihzahrd.fileutils.coordinates.Coordinates, variation_index: int, home: Optional[lihzahrd.fileutils.coordinates.Coordinates] = None)¶

A NPC somewhere in the world.

home¶

The coordinates of the home of this NPC, or None if the NPC is homeless.

name¶

The name of this NPC.

position¶

The position of the mob in the game world.

type¶

The NPC this object represents.

variation_index¶

Unsure, but seems to be an index to different possible NPC variations.

class lihzahrd.npcs.Mob(type_: int, position: lihzahrd.fileutils.coordinates.Coordinates)¶

A Mob somewhere in the world.

position¶

The position of the mob in the game world.

type¶

The type of mob this object represents.
