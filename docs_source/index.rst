lihzahrd
====================================

.. image:: lihzahrd.png
    :width: 32px
    :height: 48px
    :alt: A Lihzahrd from the Jungle Temple

``lihzahrd`` is a Terraria 1.3.5.3 world parser in Python.

You can use this package to get programmer-friendly data from a Terraria world!
